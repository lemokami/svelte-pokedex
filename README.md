# Svelte Pokedex

A simple webapp that uses PokeAPI & SvelteKit to show the deatils of pokemon.

## Installation

After cloning the repo:

Install svelte-pokedex with npm

```bash
  npm install
  npm run dev
```

Install svelte-pokedex with yarn

```bash
  yarn # you can use 'yarn install' as well
  yarn run dev
```
